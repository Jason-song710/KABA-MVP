# 화면별 기능 설명

## 사용자 화면

- 로그인: 승인된 회원사 계정만 접속
- 회원가입 신청: 회사명, 담당자명, 연락처, 회원사 유형, 관심 분야 입력
- 승인 대기: 관리자 승인 전에는 공고 목록 접근 불가
- 입찰 진행중 공고: 마감일이 지나지 않았거나 마감일이 비어 있는 공고를 기본 조회
- 오늘 등록 공고: 오늘 등록된 공고만 조회
- 자동 갱신 상태: 목록을 60초마다 다시 조회하고 마지막 갱신시각 표시
- 핵심공고: 최종 분류가 `주소산업 핵심공고`인 공고 조회
- 관련공고: 최종 분류가 `주소산업 관련공고`인 공고 조회
- 참고공고: 최종 분류가 `참고공고`인 공고 조회
- 전체: 제외공고까지 포함한 전체 조회
- 키워드 검색: 공고명, 발주기관, 상세내용 검색
- 공고 상세보기: 발주기관, 공고일, 마감일, 예산, 원문 URL, 상세내용, 첨부파일 URL 확인
- AI 요약 보기: AI 판단 또는 공고 상세 기반 요약 확인
- 분류 사유 보기: AI 판단 사유, 매칭 산업, 추천 회원사 유형, 리스크 메모 확인

## 관리자 화면

- 회원 관리: 가입 신청, 승인 회원, 반려/비활성 회원을 조회하고 회사 정보, 추천 키워드, 승인 상태를 수정하거나 탈퇴 처리
- 나라장터 수집: 기간과 AI 적용 여부를 지정해 API 전체 수집 실행. 용역/물품/공사/기타/외자 공고의 최근 등록과 마감/개찰 기준 조회를 함께 수행하고 신규/갱신/중복 건수를 표시
- CSV 업로드: API 장애 또는 별도 확보 데이터를 CSV로 업로드
- 공고별 1차 점수 확인: 목록과 상세 패널에서 점수, 후보 분류, 매칭 키워드 확인
- AI 2차 분류 결과 확인: 최종 분류, AI 점수, 사유, 산업 태그 확인
- 분류 수정: 관리자가 최종 분류와 수정 사유 저장
- 수동 재분류: 1차 분류 또는 AI 포함 재분류 실행
- 키워드 추가/삭제: S/A/B/C/D 등급 키워드 관리
- 제외 키워드 관리: 제외 키워드 추가/삭제
